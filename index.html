⚙️ Önce kurulum gerekli

Uygulamanın çalışması için index.html içindeki AYARLAR bloğunu doldurmalısın.

  1. Supabase projesi aç → Project Settings → API'den Project URL ve anon key'i kopyala.
  2. schema.sql dosyasını Supabase → SQL Editor'de çalıştır.
  3. Google Sheets'te apps-script.gs'i web uygulaması olarak yayınla, /exec URL'sini al.
  4. Bu üç değeri index.html en üstteki CONFIG alanına yapıştır ve sayfayı yenile.
